# EliteStore — Premium E-commerce Website

## 🚀 Project Overview

**EliteStore** is a modern, responsive, and elegant e-commerce front-end built with:
- Vite
- React + TypeScript
- Tailwind CSS
- Radix UI
- Shadcn UI
